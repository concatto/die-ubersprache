O projeto deve ser importado no Eclipse. O último teste foi realizado na versão Oxygen.3a Release (4.7.3a, build 20180405-1200)

Para importar:
1. File -> Import -> Existing projects into workspace -> Select root directory -> Browse;
2. Selecione a pasta New, presente na raiz do projeto enviado;
3. Clique em Finish.


Caso haja algum problema com o JavaFX (diagnóstico: problemas nos imports da classe application.Controller):

1. Help -> Install New Software -> Work with;
2. Selecione a opção "Oxygen - http://download.eclipse.org/releases/oxygen";
3. Filtre ("type filter text") com o termo "e(fx)clipse", com parênteses e sem aspas;
4. Selecione o pacote "e(fx)clipse - IDE".
5. Siga o processo de instalação, aceitando os termos quando necessário.



Os exemplos de uso da linguagem podem ser encontrados no arquivo "control_flow.txt" na raiz do projeto.



Operadores relacionais:
- < se torna "kleiner als" (menor que)
- > se torna "grösser als" (maior que)
- <= se torna "kleiner oder gleich als" (menor ou igual a)
- >= se torna "grösser oder gleich als" (maior ou igual a)
- == se torna "gleich" (igual)
- != se torna "ungleich" (diferente)



Estruturas de controle:
- if se torna "falls"
- else se torna "sonst"
- while se torna "während"
- do se torna "tun"
- for se torna "für"
